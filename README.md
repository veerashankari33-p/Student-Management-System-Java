# Student Management System (Java)

## Overview
A simple console-based Student Management System developed using Java. This project allows users to manage student records through CRUD operations.

## Features
- Add Student
- View Students
- Update Student
- Delete Student
- Menu-driven Console Application

## Technologies Used
- Java
- OOP (Object-Oriented Programming)
- ArrayList
- Scanner

## Project Structure

```
src/
 ├── Main.java
 └── Student.java
```
